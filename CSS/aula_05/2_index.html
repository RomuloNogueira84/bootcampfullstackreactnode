O dia em que estou aprendendo CSS

Hoje dia 28 de outubro de 2024

O professor Gabriel Braga me ensinou sobre CSS, e eu não estava preparado para isso. Eu estava aprendendo HTML e JavaScript, mas não sabia o que era CSS. Mas, como eu estava aprendendo sobre HTML e JavaScript, eu percebi que o CSS era algo muito importante para mim. Então, fui aprendendo sobre CSS e me sinto muito feliz por ter aprendido.

Hoje, eu estou aprendendo sobre seletores, propriedades, valores e pseudo-elementos. Eu estou aprendendo sobre a sintaxe básica do CSS e como ele funciona. Eu também estou aprendendo sobre o uso de variáveis e funções em CSS. E, finalmente, estou aprendendo sobre layouts e grids.

Eu estou aprendendo sobre o uso de pseudo-elementos, como eles são usados para adicionar efeitos especiais a elementos HTML. Eu também estou aprendendo sobre o uso de media queries, que são usados para ajustar o layout de um site para diferentes dispositivos e tamanhos de tela.
